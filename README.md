# Raspberry PI Pico
A demo for the Raspberry Pi Pico W using .Net nanoFramework.

Before being able to run/deploy/debug the app, the `Secrets.cs.template` needs to be renamed to `Secrets.cs` and the correct information added.

For further information on how to flash the CLR and other features, see https://github.com/nanoframework/nf-interpreter/blob/main/targets/ChibiOS/RP_PICO_RP2040/README.md
